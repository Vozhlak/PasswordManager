## 🔍 Описание

**Password Manager** — это безопасное консольное приложение для управления паролями, разработанное на языке Go. Все данные шифруются с использованием алгоритма **AES-256-GCM**, что гарантирует конфиденциальность и целостность информации.

### Ключевые особенности:
- 🔐 **Мастер-пароль**: единый ключ доступа ко всем данным
- 🛡️ **AES-GCM шифрование**: аутентифицированное шифрование с защитой от подмены
- 🎲 **Генератор паролей**: создание криптостойких паролей заданной длины
- 📊 **Статистика и аналитика**: категории, дубликаты, временные метки
- 🎨 **Цветной CLI-интерфейс**: удобный и наглядный вывод
- 💾 **Автосохранение**: данные сохраняются при выходе из приложения

## ⚙️ Функциональность

### Управление паролями
| Функция | Описание |
|---------|----------|
| `Добавить пароль` | Сохранение нового пароля с именем, значением и категорией |
| `Получить пароль` | Поиск и отображение пароля по имени сервиса |
| `Обновить пароль` | Изменение существующего пароля с валидацией сложности |
| `Удалить пароль` | Безвозвратное удаление записи из хранилища |
| `Список паролей` | Табличный вывод всех сохранённых паролей |

### Безопасность
| Функция | Описание |
|---------|----------|
| `Мастер-пароль` | Минимум 8 символов, используется для генерации ключа шифрования |
| `AES-256-GCM` | Современный режим шифрования с аутентификацией данных |
| `Уникальный nonce` | Случайный вектор инициализации для каждого сохранения |
| `Валидация паролей` | Проверка сложности: регистр, цифры, спецсимволы, запрет пробелов |

### Утилиты
| Функция | Описание |
|---------|----------|
| `Генератор паролей` | Создание случайных паролей длиной от 8 до 128 символов |
| `Поиск дубликатов` | Выявление паролей, используемых в нескольких сервисах |
| `Категории` | Группировка паролей по тегам (работа, личное, соцсети) |
| `Статистика` | Общее количество, распределение по категориям, хронология |

| Компонент | Версия |
|-----------|--------|
| **Go** | 1.20 или выше |
| **OS** | Linux / macOS / Windows |
| **Память** | 64 MB RAM |
| **Диск** | 10 MB свободного места |

### Зависимости
```bash
# Внешняя библиотека (устанавливается автоматически при сборке)
go get golang.org/x/term
```

## 📦 Установка
### Способ 1: Сборка из исходников (рекомендуется)
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Vozhlak/PasswordManager.git
cd passwordManager

# 2. Установите зависимости
go mod tidy

# 3. Соберите исполняемый файл
go build -o pm main.go

# 4. Запустите
./pm
```

### Способ 2: Установка через `go install`
```bash
go install github.com/Vozhlak/PasswordManager@latest
# Исполняемый файл появится в $GOPATH/bin или $HOME/go/bin
```

### Способ 3: Готовый бинарник (для релизов)
```bash
# Скачайте архив для вашей ОС с GitHub Releases
wget https://github.com/Vozhlak/PasswordManager/releases/download/v1.0.0/pm-linux-amd64.tar.gz

# Распакуйте и запустите
tar -xzf pm-linux-amd64.tar.gz
./pm
```
